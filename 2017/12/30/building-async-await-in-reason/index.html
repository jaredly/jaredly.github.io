Making it general

How do we distinguish between when we want await to be for Promises vs Observables?

My macro is based heavily on ppx_let from janestreet, and I copied their convention in this case. [%await let x = y] transforms into Let_syntax.bind(y, x => ..., so it assumes that you have defined what Let_syntax is. For Promises, you do let module Let_syntax = Reason_async.Promise, and it would work similarly for observables (I haven‚Äôt implemented that yet).
